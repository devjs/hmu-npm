# hmu-npm
> A hmu plugin for checking npm package availability

This is a plugin for [`hmu`][hmu] to check npm package name availability on the [npm registry][npm].

## Installation
```shell
$ npm install -g hmu-npm
```

## Usage
```javascript
$ hmu npm <...packages>
```
