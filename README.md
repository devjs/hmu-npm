# Hit Me Up for npm
> Hit me up with npm package name availability.

This is a [Hit Me Up][hmu] plugin for checking npm package name availability.

## Installation
```shell
$ npm install --global hmu-npm
```

## Usage
```shell
$ hmu npm <...names>
```
